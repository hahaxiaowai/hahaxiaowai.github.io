页面导航

border-box ​

css
* {
  box-sizing: border-box;
}

设置元素的宽高包含padding和border，默认是content-box，即宽高不包含padding和border

可以避免设置padding和border后元素宽高发生变化，导致布局混乱的问题，在实际开发中，建议全局设置box-sizing: border-box;，这样可以更方便地控制元素的尺寸和布局。

label标签 ​

label标签用于定义表单控件的标签，点击label标签会自动聚焦到对应的表单控件上，提升用户体验。使用label标签时，可以通过for属性将label标签与表单控件关联起来，for属性的值应该与表单控件的id属性值相同。例如：

html
<label for="username">用户名：</label>
<input type="text" id="username" name="username">

在这个例子中，label标签的for属性值为"username"，与input标签的id属性值相同，这样当用户点击"用户名："这个标签时，输入框会自动获得焦点，方便用户输入。

form button reset ​

form标签中，button标签的type属性默认为"reset"，表示点击按钮会重置表单数据。

雪碧图 ​

雪碧图是一种将多个小图标合并成一张大图的技术，可以减少HTTP请求，提高页面加载速度。使用雪碧图时，可以通过CSS的background-position属性来显示不同的图标。例如：

css
.icon {
  background-image: url('sprite.png');
  width: 32px;
  height: 32px;
}
css
.icon1 {
  background-position: 0 0;
}

这样就可以显示雪碧图中的第一个图标，其他图标可以通过调整background-position的值来显示。使用雪碧图可以有效地优化页面性能，减少加载时间。 如果想展示第二排第一个，将background-position设置为0 -32px；第三排第一个，设置为0 -64px，以此类推。

绝对定位 ​

绝对定位是指元素相对于最近的定位祖先元素进行定位，如果没有定位祖先元素，则相对于初始包含块进行定位。使用绝对定位时，可以通过CSS的position属性设置为absolute，并使用top、right、bottom、left属性来指定元素的位置。

z-index属性可以控制元素的堆叠顺序，数值越大，元素越靠前显示。使用绝对定位时，可以通过设置z-index属性来控制元素的层级关系。

如果二者结合使用，需要注意，绝对定位元素的z-index值必须大于相对定位元素的z-index值，才能确保绝对定位元素在相对定位元素之上显示。

属性值的计算过程 ​

属性值的计算过程是指浏览器在渲染页面时，如何计算元素的最终样式。这个过程包括以下几个步骤：

  1. 解析HTML和CSS，构建DOM树和CSSOM树。

  2. 计算每个元素的样式，生成渲染树。

  3. 布局渲染树，计算每个元素的位置和大小。

  4. 绘制渲染树，将元素绘制到屏幕上。 在这个过程中，浏览器会根据CSS的层叠规则和继承规则来计算元素的样式，最终生成渲染树。布局阶段会根据元素的尺寸和位置来计算每个元素的最终位置和大小，最后绘制阶段会将元素绘制到屏幕上，完成页面的渲染。

  5. 确定声明值：参考样式表中，没有冲突的声明，作为属性值

  6. 层叠冲突，对有冲突的声明，按照优先级规则选择一个声明，作为属性值

  7. 使用继承值，如果属性没有声明值，并且是可继承的属性，则使用父元素的属性值，作为属性值

  8. 使用初始值，如果属性没有声明值，并且不是可继承的属性，则使用属性的初始值，作为属性值 举例：如何让a标签字体颜色跟随父元素颜色

css
.parent {
  color: red;
}
.parent a {
  color: inherit;
}

在这个例子中，a标签的color属性设置为inherit，表示继承父元素的颜色，这样a标签的字体颜色就会跟随父元素的颜色变化。 text-decoration属性同样可以设置为inherit，表示继承父元素的文本装饰，例如下划线、删除线等。也可以设置为none。

无论单位写的是px、em、rem，浏览器都会将其转换为实际的像素值进行渲染。例如，1em等于当前元素的字体大小，1rem等于根元素的字体大小，10px就是10像素。浏览器会根据这些单位的定义来计算元素的尺寸和位置，从而实现页面的布局和样式。

伪类选择器 ​

伪类选择器是CSS中的一种选择器，用于选择元素的特定状态或位置。常用的有：

  • :first-child：选择父元素的第一个子元素。
  • :last-child：选择父元素的最后一个子元素。
  • :nth-child(n)：选择父元素的第n个子元素，n可以是一个数字、关键词或公式。
  • :nth-last-child(n)：选择父元素的倒数第n个子元素。
  • :first-of-type：选择父元素的第一个指定类型的子元素。
  • :last-of-type：选择父元素的最后一个指定类型的子元素。
  • :nth-of-type(n)：选择父元素的第n个指定类型的子元素。
  • :nth-last-of-type(n)：选择父元素的倒数第n个指定类型的子元素。
  • :not(selector)：选择不匹配指定选择器的元素。
  • :hover：选择鼠标悬停在元素上的状态。
  • :active：选择元素被激活（例如被点击）时的状态。
  • :focus：选择元素获得焦点时的状态。
  • :disabled：选择被禁用的元素。
  • :checked：选择被选中的输入元素（例如复选框或单选按钮）。
  • :empty：选择没有子元素的元素。
  • :root：选择文档的根元素。
