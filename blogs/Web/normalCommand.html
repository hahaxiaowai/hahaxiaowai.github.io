常用库的命令 ​

简述 ​

记录个人常用的库的命令行

npm-check-updates(ncu) ​

检测npm包是否需要更新

js
// 检测xx是否有最新版本
// 不加则检测当前package.json所有的包
ncu xx
// 更新xx在package.json的版本号
// 不写则更新所有
ncu -u xx

lerna ​

包管理器

js
// 添加依赖库xx到yy,scope为作用范围，不设置则为整个环境
lerna add xx --scope=yy
// 安装依赖库(npm install)
lerna bootstrap
// 创建并初始化包文件夹xx
lerna create xx

nvm ​

window安装需要去官网下载

js
// 查看当前的node版本列表
nvm ls
// 使用某一版本
nvm use XX.XX.X
// 下载
nvm install XX.XX.X

rimraf 快捷删除 node_modules 文件夹 ​

js
rimraf node_modules

http-server 开https服务 ​

在项目目录中打开git bash 输入命令生成证书

js
openssl req -newkey rsa:2048 -new -nodes -x509 -days 3650 -keyout key.pem -out cert.pem

输入命令后会提示输入一些信息，除邮箱外全填zh，邮箱乱写就行，保证邮箱格式， 之后在项目中运行http-server命令

js
http-server -S -C cert.pem -o
