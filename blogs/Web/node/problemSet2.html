excel-export ​

可做数据导出成excel，相应的有excel-parser用来解析excel数据。 该库不能使用中文作为sheet的名字，否则会报错

数据格式：

js
const test = {
      name: "name",
      cols: [
        {
          caption: "ID",
          type: "string",
        },
        {
          caption: "model_name",
          type: "string",
        },
      ],
      rows: [
        ["1", "test"],
        ["2", "test"],
      ],
    };
    // 写入
    fs.writeFile(`data/res.xlsx`, nodeExcel.execute(test), "binary", (err) => {
      console.log(err);
    });

如果想要写多个sheet,在执行execute时以数组的形式穿进去即可

js
nodeExcel.execute([test,test2])
