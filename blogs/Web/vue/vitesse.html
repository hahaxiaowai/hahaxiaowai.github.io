简述 ​

vitesse是一个学习vite+vue3非常好的模板，也可以基于此模板制作属于自己的模板，如果要在项目中使用，建议使用nuxt.js作为基础。本篇学习其各种配置并改成js版本

vite-ssg ​

适用于官网、静态资源比较多的网站 安装

js
npm i -D vite-ssg

引入

js
import { ViteSSG } from 'vite-ssg'
export const createApp = ViteSSG(
  // the root component
  App,
  // vue-router options
  { routes,base: import.meta.env.BASE_URL },
)

路由 ​

vite-plugin-pages ​

以文件系统为基础的路由

安装

js
npm install -D vite-plugin-pages
npm install vue-router

配置

js
// vite.config.js
import Pages from 'vite-plugin-pages'

export default {
  plugins: [
    // ...
    Pages({
      extensions: ["vue", "md"], // 识别vue md文件
    }),
  ],
}

vite-plugin-vue-layouts ​

页面布局系统

js
npm install -D vite-plugin-vue-layouts

配置

js
// vite.config.js
import Pages from 'vite-plugin-pages'

export default {
  plugins: [
    // ...
    Pages({
      extensions: ["vue", "md"], // 识别vue md文件
    }),
  ],
}

引入

js
// main.js
import { setupLayouts } from 'virtual:generated-layouts'
import generatedRoutes from '~pages'

const routes = setupLayouts(generatedRoutes)

export const createApp = ViteSSG(
  App,
  { routes,base: import.meta.env.BASE_URL },
)

自动导入 ​

unplugin-vue-components ​

自动引用组件

安装

js
npm i unplugin-vue-components -D

配置

js
// vite.config.js
import Components from 'unplugin-vue-components/vite'

export default {
  plugins: [
    // ...
    Components({
      // 自动导入.vue md 组件 src/components
      extensions: ['vue', 'md'],
      // 自动引入并注册组件
      include: [/\.vue$/, /\.vue\?vue/, /\.md$/],
    }),
  ],
}

unplugin-auto-import ​

自动导入

安装

js
npm i -D unplugin-auto-import

配置

js
// vite.config.js
import AutoImport from 'unplugin-auto-import/vite'

export default {
  plugins: [
    // ...
    AutoImport({
      imports: [
        'vue',
        'vue-router',
      ],
      dirs: [],
      vueTemplate: true,
    }),
  ],
}

@unhead/vue ​

原来的@vueuse/head,现在扩展到任何框架下使用了

安装

js
npm i @unhead/vue

配置（自动导入）

js
// vite.config.js
import { unheadVueComposablesImports } from '@unhead/vue'
export default {
  plugins: [
    // ...
    AutoImport({
      imports: [
        unheadVueComposablesImports,
        '@vueuse/core',
      ],
      dirs: [],
      vueTemplate: true,
    }),
  ],
}
